This project implements a Monte-Carlo + Optimization-based Markowitz Portfolio Engine using real market data.
It builds the efficient frontier, finds the maximum Sharpe ratio portfolio, and visualizes risk-return trade-offs.

The model is built using modern portfolio theory (MPT) and is suitable for quantitative finance and asset management research.
